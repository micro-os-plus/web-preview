This site uses the GitHub Wiki-like theme by Liviu Ionescu.

First µOS++ scriptable build

Jan 22, 2021 - In preparation for the future 7.x release, a major milestone was reached: the modularised µOS++ was, for the first time, fully built without Eclipse.

µOS++ IIIe v6.3.15 released

Nov 19, 2018 - Version v6.3.15 is a µOS++ IIIe maintenance release. The main changes are several bug fixes.

µOS++ IIIe v6.3.14 released

Jul 4, 2018 - Version v6.3.14 is a µOS++ IIIe maintenance release. The main addition is the functional POSIX I/O subsystem, which supports character devices, block devices and file systems.

µOS++ IIIe v6.3.13 released

Aug 26, 2017 - Version v6.3.13 is a µOS++ IIIe maintenance release.

µOS++ IIIe v6.3.11 released

Jan 2, 2017 - Version v6.3.11 is a new µOS++ IIIe/CMSIS++ public release that addresses several bugs.

µOS++ IIIe v6.3.10 released

Nov 27, 2016 - Version v6.3.10 is a new µOS++ IIIe/CMSIS++ public release that addresses several bugs.

µOS++ IIIe v6.3.9 released

Oct 13, 2016 - Version v6.3.9 is a new µOS++ IIIe/CMSIS++ public release (v6.3.8 was an internal release). The major additions are: the first integration of the POSIX I/O subsystem and an experimental set of scripts to generate µOS++ projects.

First µOS++ IIIe public version v6.3.7 released

Sep 15, 2016 - Version v6.3.7 is the first µOS++ IIIe/CMSIS++ public release. The major additions are the advanced memory management features, which allow deterministic and non-fragmenting allocations in controlled environments.

License changed from LGPL to MIT License

Mar 29, 2016 - To further facilitate the integration of CMSIS++ files into other commercial or open source projects, the project license was changed from the copyleft LGPL to the permissive MIT License.

ARM Connected Community article, March 11, 2016

Mar 10, 2016 - A presentation article was published on the ARM Connected Community site, under the name CMSIS++: a proposal for a future CMSIS, written in C++.

The µOS++ IIIe new web site (using GitHub Pages)

Mar 3, 2016 - After the µOS++ IIIe project was migrated from SourceForge to GitHub, a new web site was created, to replace the previous wikis.
